# STATUS

## Project

Not every stable cluster is homeostatic: Control, projectibility, and the Boyd--Khalidi divide.

## Stage

Seed project scaffolded on 2026-05-28. Public GitHub repo created as infrastructure for a short philosophy-of-science article.

## Working Thesis

Boydian HPC theory needs a sharper homeostasis constraint. Khalidi's causal-network theory supplies internal order and projectibility, but network order is not the same as homeostatic control. Weinberger's control-theoretic account allows a three-way distinction:

1. stable property clusters;
2. network-ordered projectible kinds;
3. strict homeostatic-control kinds.
